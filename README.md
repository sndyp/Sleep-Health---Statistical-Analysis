# Sleep Health and Lifestyle Analysis 🌙📊

## Overview
This project performs a comprehensive analysis of the **Sleep Health and Lifestyle Dataset** to understand the factors influencing sleep quality and duration. By examining variables such as physical activity, stress levels, BMI, and daily habits, the study provides insights into how lifestyle choices impact overall well-being.

## Project Structure
- `FinalReport_Group4.pdf`: Complete analytical report including methodology and findings.
- `FinalReport_Group4.Rmd`: R Markdown source code for the analysis.
- `explorationFunction.R`: Custom R functions developed for data exploration and summary.
- `SleepDataset.csv`: The raw dataset used for analysis.
- `Infografis AOL DMV.jpg`: Visual summary/infographic of the key insights.

## Key Insights
- **Stress & Sleep:** Analysis of how stress levels correlate with sleep quality across different occupations.
- **Physical Activity:** Impact of daily steps and physical activity levels on resting heart rate and sleep duration.
- **Health Indicators:** Exploration of BMI categories and blood pressure in relation to sleep disorders like Insomnia and Sleep Apnea.

## Technologies Used
- **Language:** R
- **Key Libraries:** `ggplot2`, `dplyr`, `knitr` (and custom exploration functions).
- **Documentation:** R Markdown & PDF.
